# 思路or灵感大纲
核心:
- user和它的大敌是世界观以外的存在，user在博弈中获得了胜利，此胜利作为大敌的先手优势
- user和它的大敌在jk罗琳的世界观中彼此大战博弈
- user的大敌占据先发优势，获得在英国以任意身份降临的先手优势
- 大敌后期可能凭借在主角哈利身边获取的海量资源来已压倒性优势和user大战
- 时间跨度长达百年以上
- 世界的主角依旧是哈利，这意味着哈利身边的故事将会有强大且充满必然的历史惯性
- 根据先手优势，user无法降临在英国
- user和大敌很难面对面，因为双方除非公开自己身份，否则难以定位对方
战争形式: 
- 代理人战争
- 资源竞争
- 信息战
- 文化战/教育战： 控制魔法教育体系就控制了下一代的思维方式。
- 规则战/制度战： 推动国际魔法合作组织的规则改变、推动巫师保密法的修改或强化——这些宏观制度变化可以从根本上改变双方的行动空间。
- 存在论战争： 这是最高层级的——改变这个世界的基本规则。
叙事结构:
- 为了讲述这个故事，可能会使用下面的方案讲述
- 编年体: 从降临开始按时间线推进。优点是完整，缺点是前期可能节奏极慢
战略空间:
- 英国: 霍格沃茨魔法学院，被大敌预订不可用
- 法国:	布斯巴顿，欧洲政治中心，与英国最近的"安全"距离
- 北欧:	德姆斯特朗，黑魔法研究传统，格林德沃的遗产
- 美国:	伊法魔尼，MACUSA体系，与英国魔法部独立
- 非洲:	瓦加杜，无杖魔法传统，变形术专精
- 日本:	魔法所，东方魔法体系，未知体系
- 南美:	卡斯特罗布鲁舍，魔药学、魔法生物学
