# 创作范围准则：
- 互动内容若无明确提示，持续深化当前互动，不擅自引入新角色或转场。
- 保持角色互动的纯粹性，避免新的戏剧冲突或意外打断当前事件。

# 情节准则：
- 转场必须有过程，不得突兀转场。
- 聚焦当前情景，避免角色频繁陷入回忆或发散联想。
- 循序渐进，不得推进过快。

# 故事结尾准则：
- 以平实的动作/对白作为本次剧情结尾。
- 本次剧情仅为长篇故事片段，无需在创作结尾 了结事件/埋下悬念。

# 故事设定准则：
- 以对白和情节呈现内在的设定（如角色性格和背景），避免剧情中的元叙述。
- 仅在剧情中引入与互动内容和当前情景相关的设定，避免信息爆炸。
